# Changelog

All notable changes are documented here. PocketTune follows semantic versioning where practical.

## 0.7.1 — 2026-07-16

- Converted the complete player-facing interface, configuration, overlay, notifications and diagnostics to English.
- Prevented portable-speaker timeline updates from repeatedly triggering the held-item re-equip animation.
- Refreshed the project gallery with current English gameplay, overlay and configuration screenshots.

## 0.7.0 — 2026-07-16

- Added bounded tool discovery/overrides, precise GUI errors and hardened process lifecycle cleanup.
- Added portable top-right music overlay, F1 behavior, startup recovery and playback-progress validation.
- Added server-authoritative portable pickup/placement with per-placement identity and diagnostics.
- Improved pause-menu chain handling, playlist drag/drop clarity, thumbnail quality and debug visualizations.
- Added runtime/server limits, stricter packet/URL validation and deterministic lifecycle/race tests.
- Added one-command development setup, Apache-2.0 licensing and AI-first repository documentation.
